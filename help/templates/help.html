Introduction


    Statement: All the datasets on the server are computed by a standard pipeline . Use the Google Chrome browser for best visualization quality.

    GEIMA is a web server for query Gene Expression in different GSE data from the NCBI, using a standard processing pipeline.


Quick start

Parameters

  • Disease Type: Select a disease type.
  • Species Type: Select a species type.
  • Tissue Select: Select a tissue type.

  • Query GSE database: Select GSE database.
  • Input Gene Symbol: Input a official gene symbol.

  • All DEGs result download: download the difference analysis results of gene in all GEO datasets.
  • Filter DEGs result download: download the difference analysis results of gene with |log2Foldchange|>log2(1.5) and P value<0.05.
  • Expression result download: download the expression results of gene in all GEO datasets.

Search

  • Input official gene symbol in the "Input Official Gene Symbol" field, chose disease type,species type,tissue type ，and click the "Query"button to search for the gene of intrest.
  • If you have multiple genes, you can switch to the "Muliple Genes" mode and paste the gene list to the "Input Official Gene Symbol" field.
  • If there are too many genes, you can switch to the "Gene List" mode and upload the gene list file (The gene list file must be in text file format).
  • The gene list file template.
  • Click the "Query GSE database" button can view the GEO datasets contained in the current database

Results

  • Barplot(only show in single gene mode)

      • Barplot shows the Foldchange of gene in the top 20 GEO datasets

  • Table

      • Table shows the difference analysis results of gene in different GEO datasets
